# Differentiating between real and fake images using texture-based features
To test:
- download model from https://drive.google.com/file/d/12bs5I4lcK0PV7B9WELXpfpGv_ZOZHHKX/view?usp=drive_link and move it into the root dir
- For comparisons of generators:
  - download this dataset https://www.kaggle.com/datasets/jayanthbottu/labeled-deepfake-image-collection/data and name the folder data
  - Then, run python test_by_generator.py
- To test with a different dataset than the one the model was trained on:
  - download this dataset https://www.kaggle.com/datasets/ayushmandatta1/deepdetect-2025/data
  - Then, run python test.py
